## 1. 개요

## 2. 서브명령어

| 명령어 | 설명 | 출력 |
|--------|------|------|
| `oaisuser run` | 신규 생성 | doc/d{SP}0008_user.md |
| `oaisuser status` | 서브명령어 리스트, 현황 조회 | 터미널 |
| `oaisuser version` | 스킬 버전 정보 (v01) | 터미널 |
| `oaisuser add [기능명]` | 기능 사용법 추가 | doc/d{SP}0008_user.md |
| `oaisuser faq [질문]` | FAQ 추가 | doc/d{SP}0008_user.md |
| `oaisuser sync` | PRD 기반 동기화 | doc/d{SP}0008_user.md |

```markdown

접속 → 로그인 → 기본 사용법

## 6. 트러블슈팅
```

## 4. 워크플로우

| 명령어 | 흐름 |
|--------|------|
| run | PRD 수집 → 템플릿 생성 → 저장 |
| add | 기능 수집 → 템플릿 적용 → 추가 |
| sync | PRD↔가이드 비교 → 동기화 보고 |
| faq | 카테고리 결정 → FAQ 추가 |
